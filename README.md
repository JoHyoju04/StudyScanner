# StudyScanner
고객에게 실시간 좌석현황을 제공하여 스터디 카페에 직접 방문하지 않아도 남아있는 좌석을 알 수 있도록 하여
시간과 공간의 제약없이 원하는 날짜와 시간에 개인좌석 및 스터디룸을 예약할 수 있는 기능을 제공하는 웹페이지이다.
